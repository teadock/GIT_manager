# GitHub Repository Manager - Requirements

# No Python packages required!
# All dependencies are in Python's standard library:
# - tkinter (GUI framework - usually pre-installed)
# - subprocess (for running gh commands)
# - json (for parsing API responses)
# - os, platform, datetime (system utilities)

# External Requirements:
# 1. GitHub CLI (gh) - Install separately:
#    Windows: winget install --id GitHub.cli
#    macOS: brew install gh
#    Linux: sudo apt install gh (or sudo dnf install gh)
#
# 2. Authentication:
#    gh auth login (select SSH protocol!)
#    gh auth refresh -h github.com -s delete_repo


# External dependency (not Python):
# - GitHub CLI (gh) - Install separately:
#   macOS: brew install gh
#   Windows: winget install GitHub.cli
#   Linux: See https://cli.github.com/

# Python version requirement:
# Python >= 3.8

# Note: Tkinter comes with Python on macOS and Windows.
# On Linux, install separately:
#   Ubuntu/Debian: sudo apt install python3-tk
#   Fedora/RHEL: sudo dnf install python3-tkinter
